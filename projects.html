Medtronic Smart Bathroom Project

Collaborating with a four person St. Thomas team and Medtronic engineers on a medical device system, exploring both deep learning and classical signal processing approaches; developing RNN architectures in PyTorch alongside traditional algorithms for potential deployment.

IoT Sensor Integration Project

Programmed bare-metal C firmware with GPIO interrupts for keypad scanning, SPI for LCD control, and timer-based PWM for LED effects; implemented debouncing algorithms and ISRs for real-time input handling.
